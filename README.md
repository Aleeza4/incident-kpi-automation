# Incident KPI Automation System

Python-based ITSM incident reporting and KPI automation project.

## Features
- Automated KPI calculation
- Incident data cleaning
- High-priority incident filtering
- Excel report generation
- Automated chart generation
- Monthly incident trend analysis

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- OpenPyXL

## Generated Reports
- cleaned_incidents.xlsx
- incident_summary.xlsx
- high_priority_incidents.xlsx

## Generated Charts
- Incident Type Distribution
- Priority Distribution
- Monthly Incident Trend
- Assignment Group Analysis

## Business Use Case
This project automates repetitive ITSM operational reporting tasks and helps operations teams reduce manual reporting effort.
